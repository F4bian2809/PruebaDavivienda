# PruebaDavivienda

Repositorio público para pruebas técnicas y demostraciones.

## Descripción

Este proyecto parece estar orientado a realizar una prueba técnica, posiblemente para la compañía Davivienda, y contiene recursos en los lenguajes HTML, Java, JavaScript y Gherkin. Sin una descripción explícita en el repositorio, asumimos que el objetivo es demostrar habilidades de desarrollo y/o automatización de pruebas.

## Tecnologías y Lenguajes Utilizados

**HTML** (mayoría del código)
**Java** (automatización o backend)
**JavaScript**
**Gherkin** (para definir escenarios de testing tipo BDD)

## Estructura del Proyecto

El proyecto cuenta con archivos relevantes para desarrollo web y pruebas automatizadas. Incluye archivos y/o carpetas escritos en los lenguajes mencionados.

## Instalación & Uso

Dado que no existe documentación específica, aquí tienes una guía general:

1. **Clona el repositorio:**
  
bash
   git clone https://github.com/F4bian2809/PruebaDavivienda.git
   cd PruebaDavivienda
  

2. **Revisa los requisitos de cada tecnología**:
   - **Java:** Asegúrate de tener instalado Java JDK 8+.
   - **Gherkin:** Usualmente utilizado con frameworks como Cucumber.
   - **HTML/JavaScript:** Solo necesitas un navegador para visualizar archivos .html.

3. **Ejecución**:
   - Abre los archivos HTML en tu navegador para ver la parte visual.
   - Para pruebas automáticas o ejecución de Java/Gherkin, sigue la estructura típica de proyectos Java (puedes emplear Maven o Gradle si existe un archivo de configuración).
